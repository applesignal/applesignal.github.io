# 一鍵翻譯工具

## 功能

支持單行，lua語言檔，json語言檔的翻譯

## 收費方式

```md
行數收費
100TWD = 1000行
1000TWD = 15000行
```
```md
時間收費
2500TWD = 永久無限行數
```

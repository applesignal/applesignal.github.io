# vehic

## 簡述

為影響性能、操控性和美觀性的車輛製造零件。零件磨損，需要機械師維修。允許玩家升級零件以滿足他們的駕駛需求
- 創建每個車輛零件並更改處理
- 可變引擎聲音
- 獨家配件
- 手動傳輸
- 限制工具箱、機械工具、車間的訪問權限
- Nitro套件零件
- 移動車輛信息 (里程、檢查發動機、低油位)
- 車輛零件磨損，需要維修或更換
- 可變磨損率
- 使用相對值或絕對值進行更改處理

:::tip 技工地點
891.5999755859375, -2113.510009765625, 30.45999908447265
:::

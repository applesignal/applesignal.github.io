# evide

## 簡述

證據系統非常適合警察和強盜的情況！您可以設置一個等級，官員可以根據該等級檢查車內發現的血跡、子彈殼和指紋。添加警探以互動方式分析現場並根據證據報告創建整個調查！
- 戴手套不留指紋！
- 證據歸檔系統（Archive old evidence）
- 犯罪分子可以自己收集證據（僅在一段時間後！）
- 雨水沖走證據（內部證據不受影響）

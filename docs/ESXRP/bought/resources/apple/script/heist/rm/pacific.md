# pacific

## 簡述

太平洋搶劫案
- 指紋黑客小遊戲
- 打開金庫門時的警察警報
- 開始和結束搶劫時的 ```Discord``` 日誌
- 每次搶劫都有不同的大贓物  (鑽石、黑豹、pogo專用項鍊和專用瓶)
- 可以和你的朋友一起搶劫

:::tip 開始搶劫位置
-622.70, 328.733, 85.3010
:::
:::tip 完成搶劫位置
-1228.0, -662.41, 39.3575
:::
:::tip 清理指令
/pdpacific
:::

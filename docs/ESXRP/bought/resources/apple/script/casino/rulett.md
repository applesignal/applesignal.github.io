# rulett

## 簡述

- 同步的坐姿動畫
- 同步投注對象
- 比賽開始時您可以離開牌桌，比賽將繼續進行，您的投注將保持不變，如果您贏了，您將獲得獎勵
- 輸或贏時同步反應動畫
- 當您坐著或下注等時，Peds 很健談

## 配置賭桌
```jsx title="apple_rulett/config.lua"
Config.RulettTables
```

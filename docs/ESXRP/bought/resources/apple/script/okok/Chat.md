# Chat

## 簡述

- 啟用/禁用任何命令
- 更改命令
- 更改圖標
- 隱藏聊天 ```(/hide)```
- 即使聊天被隱藏，也可以選擇繼續顯示服務器/員工消息
- 在消息上顯示玩家 ID
- 設置玩家 ID 是應該向所有人顯示還是僅對服務器管理員顯示
- 清除聊天
- 任何玩家都可以清除其聊天 ```（/clear）```
- 服務器管理員可以清除全局聊天 ```(/clearall)```
- 不和諧日誌
- 超時系統 ```（mute/unmute玩家輸入聊天）```
- 私信系統
- 自動消息系統 （每 X 分鐘發送一次服務器消息）
:::tip 玩家指令
/me
/do
/try
:::

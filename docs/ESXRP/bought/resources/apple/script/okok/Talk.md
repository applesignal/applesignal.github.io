# Talk

## 簡述

- 相機根據NPC的位置自動定位和旋轉
- 在每個 NPC 中最多添加 6 個選項/動作 (執行事件)
- 設置每個NPC的名稱/對話信息/模型
- 設置可以與某個NPC互動的職業

# Market

## 簡述

- 車輛/物品/黑市
- 搜索/排序/放置廣告
- 從最新到最舊/最低價格/最高價格對項目進行排序
- 刷新頁面以尋找新廣告
- 查看賣家信息（姓名、電話號碼、產品描述）
- 取消活動廣告
- 黑市只能通過某些工作進入
- 只在黑市上使用髒錢
- 阻止某些物品進入黑市
- 阻止某些車輛在車輛市場上銷售

# tennis

## 簡述

挑戰你的朋友或敵人來一場網球 比賽！所有法院都支持，即使有多個法院彼此相鄰。您甚至可以組織一場盛大的錦標賽！

## 功能

- 完整的動畫支持 - 不同的擊打、弓步和跳躍
- 支持所有網球場
- 全面的聲音支持
- 很好玩，容易上手，高技能上限

# rcore

## 簡述

遊戲機中心,可以遊玩 ```吃豆人``` ```俄羅斯方塊``` ```乓``` ```毀滅戰士``` ```毁灭公爵3D``` ```德軍總部3D``` 等等小遊戲

:::tip 位置
590.312, 2747.5, 41.908
:::

```jsx title="config.lua"

Config.GamingMachine
```

# derby

## 簡述

啤酒、女孩、汽車、轟鳴的引擎、響亮的音樂。每輛汽車德比的甜蜜基礎。你想在汽中挑戰你的朋友或敵人嗎？現在你有機會了！下注，系好安全帶，讓我們去砸車吧。六名玩家可以進入，只有一名將成為相撲德比的冠軍。

- 多虛擬世界允許玩家大量競爭和參與
- 公開選擇競技的車輛
- 顯示玩家的分數，可以修改給玩家特定分數的獎勵
- 可設定每個玩家的出生點、每場比賽的時間限制、最少/最多玩家、私人/開放競技場
- 可以修改相機效果

:::tip 指令
/minigame join derby 加入

/minigame leave 離開
:::

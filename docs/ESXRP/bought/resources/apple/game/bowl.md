# bowl

## 簡述

保齡球腳本

:::tip 地點
-141.83, -252.71, 44.0
:::

## 使用方法

開始前
```
左右 選擇球道
Enter 確認
Backspace 取消
```

開始後
```
左右 旋轉
上下 增減力量
A和D 切換視角
左鍵 拋出保齡球
Backspace 停止遊戲
```

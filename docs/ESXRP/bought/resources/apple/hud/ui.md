# UI

## 簡述

配合響應模組 [Handler](./handler) 自定義顯示每個介面的位置/形狀/顏色/主題

## 功能

- 設置單個界面元素的大小
- 設置整個界面的字體
- 設置界面主題[MOTIVE COLOR]
- 內置通知系統
- 內置車輛界面
- 拖動界面元素

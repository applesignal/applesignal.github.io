# 前置工作

## 創建數據庫
在空白的地方右鍵創建一個新的數據

![MariaDB 資料庫官方網頁](./img/mysql1.png)

輸入新的數據庫名稱，預設是輸入fivem，然後按確定

:::tip 數據庫名稱

預設輸入fivem

:::

![MariaDB 資料庫官方網頁](./img/mysql2.png)


## 導入數據庫
```md
點擊剛才創建的數據庫
點擊紅框的位置
把fivem.sql拖進空白地方
```
